# streamlit-baseball-app.ver1

Streamlitで作成した野球成績分析アプリです。  
2023年・2024年のDeNA打者データを比較し、グラフで視覚的に分析できます。

## 機能
- 年度別成績の比較
- 成績グラフ表示（Streamlit）
- CSVファイルの読み込みと処理

## 使用技術
- Python / pandas / matplotlib / streamlit
